# Sales & Demand Forecasting (FUTURE_ML_01)

This project predicts future sales using Machine Learning techniques.

## Features
- Data cleaning and preprocessing
- Time-based feature engineering
- Lag features for improved accuracy
- Forecasting using ML models
- Model evaluation (R² Score)

## Results
- Achieved R² Score: ~0.81
- Accurate future sales prediction

##  Business Use
- Inventory planning
- Demand forecasting
- Better decision-making

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

##  Conclusion
This model helps businesses forecast demand and optimize operations.
